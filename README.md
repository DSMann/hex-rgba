# HEX to RGBA

Convert HEX to RGBA

## Installation

    npm install --save hex-rgba

## Usage

    var hexToRgba = require('hex-rgba');

    var rgba = hexToRgba('#1B2B34', 40);
    // rgba return rgba(27, 43, 52, 0.4)
